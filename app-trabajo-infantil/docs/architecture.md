# Arquitectura Tecnica

## Stack

- Next.js App Router con JavaScript
- Tailwind CSS
- Backend con Route Handlers de Next.js
- Almacenamiento temporal en disco local y memoria de proceso

## Paginas

- `/`: dashboard principal
- `/datasets`: gestion de datasets
- `/datasets/[datasetId]`: detalle del dataset
- `/comparacion`: comparacion anual
- `/procesos`: estado del procesamiento
